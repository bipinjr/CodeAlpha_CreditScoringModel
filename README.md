## Overview
This project implements a Credit Scoring Model for the CodeAlpha Machine Learning Internship (Task 1). The model predicts an individual's creditworthiness using past financial data and evaluates performance using multiple classification algorithms.

## Features
- **Synthetic Dataset Generation**: Creates a realistic financial dataset with 2000 samples
- **Feature Engineering**: Includes income, debts, payment history, and derived financial ratios
- **Multiple Classifiers**: Implements and compares three algorithms:
  - Logistic Regression
  - Decision Tree Classifier
  - Random Forest Classifier
- **Comprehensive Evaluation**: Uses Precision, Recall, F1-Score, and ROC-AUC metrics
- **Visualizations**: Includes ROC curves and performance comparison charts

## Technologies Used
- Python 3.x
- pandas & numpy for data manipulation
- scikit-learn for machine learning models
- matplotlib & seaborn for visualization

## Dataset Features
- Income
- Total Debt
- Payment History (months)
- Credit Utilization
- Debt-to-Income Ratio
- Average Monthly Payment

## Results
The model achieved strong performance across all classifiers with comprehensive evaluation metrics displayed in the notebook.
